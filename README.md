# Vision Board Portfolio
Project Overview
This project is a simple personal Vision Board / Portfolio webpage built using HTML with basic internal CSS styling.
The page displays personal information such as profile details, skills, experience, projects, education, and contact information using headings, lists, and tables.
This repository is submitted as part of a web development assignment.

How to Run:
1. Download or clone the repository
2. Open the project folder
3. Open the file named **my-vision-board.html** in any modern web browser (Chrome, Edge, Firefox, etc.)

No installation or additional setup is required.

Technologies Used :
HTML5
Basic internal CSS

Features:
Personal introduction section
Skills list
Experience table
Projects description
Education table
Contact information
Simple table border styling
